# 🗳️ Tamil Nadu Election Analysis Dashboard (Power BI)

An interactive **Power BI dashboard** analyzing the **Tamil Nadu Assembly Elections (2021–2026)**. This project transforms raw election data into meaningful insights through interactive visualizations, enabling users to explore constituency performance, geographic trends, party-wise seat distribution, constituency flips, and victory margins.

---

## 📌 Project Overview

The **Tamil Nadu Election Analysis Dashboard** is designed to provide an interactive and data-driven view of Tamil Nadu Assembly Election results. Using Power BI, the dashboard presents election insights through modern visualizations and storytelling techniques, making complex electoral data easier to understand.

---

## 🎯 Objectives

- Analyze Tamil Nadu Assembly Election results.
- Compare election outcomes across constituencies.
- Identify constituencies where the winning party changed.
- Analyze close contests and victory margins.
- Explore geographic voting patterns.
- Build an interactive dashboard for data storytelling.

3. Refresh the data if required.
4. Explore the dashboard using interactive filters and slicers.
